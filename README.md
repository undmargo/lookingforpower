# MA Thesis Appendix: Looking for Power: Gender and Gaze Patterns in the Visual World Paradigm

## About
This repository contains all R scripts used for data preprocessing, statistical analysis, and visualisation in my master's thesis "Looking for Power: Gender and Gaze Patterns in the Visual World Paradigm" (University of Konstanz, 2026).

## Repository Structure

| Folder | Content |
|--------|---------|
| `/scripts` | All R scripts for analysis (numbered in execution order) |
| `/output/figures` | Example figures generated by the analysis |
| `/output/tables` | Example output tables |
